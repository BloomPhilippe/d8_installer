# installer-drupal-8

Script shell permettant d'installer Drupal 8


## Step 1

Installer drush 8

```
composer global require drush/drush:8.*
```

## Step 2

Lancer l'installation

```
sh installer-drupal-8.sh nom-du-projet nom-db
```
